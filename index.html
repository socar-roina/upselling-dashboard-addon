사용된 빅쿼리 쿼리 (직접 검증용) 펼치기
1. 제주 인앱 신규 일별 결제 / 순매출 (본문 일별표 재현)
예약과 상품 단위로 합산 후 순액 양수만 결제로 집계
zone_id로 제주만, 예약 생성일 6/05 이후로 마이그 분리
본문 제주 신규 208건 (6/05~6/15)
WITH addon AS ( SELECT reservation_id, charge_type, SUM(amount) AS net, DATE(MIN(IF(amount>0, created_at, NULL)),"Asia/Seoul") AS fc_dt FROM `socar-data.tianjin_replica.charged_info` WHERE DATE(created_at,"Asia/Seoul") >= "2026-06-05" AND charge_type IN ("addon_carseat","addon_stroller") GROUP BY reservation_id, charge_type HAVING net > 0 ), zmap AS ( SELECT zone_id, ANY_VALUE(region1) AS region1 FROM `socar-data.soda_store.reservation_v2` WHERE region1 IS NOT NULL GROUP BY zone_id ) SELECT a.fc_dt AS dt, COUNTIF(charge_type="addon_carseat") AS carseat_cnt, SUM(IF(charge_type="addon_carseat", net,0)) AS carseat_rev, COUNTIF(charge_type="addon_stroller") AS stroller_cnt, COUNT(DISTINCT a.reservation_id) AS rsv_cnt, SUM(net) AS total_rev FROM addon a JOIN `socar-data.tianjin_replica.reservation_info` ri ON ri.id = a.reservation_id LEFT JOIN zmap z ON z.zone_id = ri.zone_id WHERE z.region1 = "제주특별자치도" AND DATE(ri.created_at,"Asia/Seoul") >= "2026-06-05" GROUP BY dt ORDER BY dt
2. 결제 후 취소, 환불 예약 검증 (net=0의 정체)
순액 0 예약을 state, cancel_at, 결제~환불 시간차로 분해
전부 cancel_at 존재로 취소된 예약
WITH rsv AS ( SELECT reservation_id, SUM(amount) AS net_amount FROM `socar-data.tianjin_replica.charged_info` WHERE DATE(created_at,"Asia/Seoul") >= "2026-06-05" AND charge_type IN ("addon_carseat","addon_stroller") GROUP BY reservation_id ), zero_rsv AS ( SELECT reservation_id FROM rsv WHERE net_amount = 0 ), gap AS ( SELECT c.reservation_id, TIMESTAMP_DIFF(MAX(IF(c.amount<0,c.created_at,NULL)), MIN(IF(c.amount>0,c.created_at,NULL)), MINUTE) AS diff_min FROM `socar-data.tianjin_replica.charged_info` c JOIN zero_rsv z USING(reservation_id) WHERE c.charge_type IN ("addon_carseat","addon_stroller") GROUP BY c.reservation_id ) SELECT ri.state, COUNT(*) AS rsv_cnt, COUNTIF(ri.cancel_at IS NOT NULL) AS has_cancel_at, ROUND(AVG(g.diff_min),1) AS avg_refund_min FROM gap g JOIN `socar-data.dbt.stg_tianjin_replica__reservation_info` ri ON ri.id = g.reservation_id GROUP BY ri.state ORDER BY ri.state
3. 구매 전환 깔대기 단계별 순 사용자
COUNT(DISTINCT IF())를 한 쿼리에 여러 개 쓰면 값이 부풀려져, GROUP BY로 단계별 집계
추가 버튼 828, 옵션 화면 785, 선택 완료 578
SELECT CASE WHEN page_name='checkout_reservation' AND component_name='custom_option_add_button' THEN '1_추가버튼' WHEN page_name='services_option' AND event_name='view' THEN '2_옵션화면' WHEN page_name='services_option' AND component_name='cta_button' THEN '3_선택완료' WHEN page_name='services_option' AND component_name='product_detail_help' THEN '4_도움말' END AS step, COUNT(DISTINCT member_id) AS uu FROM `socar-data.app_web_log.socar_app_web_log` WHERE event_date_kst BETWEEN "2026-06-05" AND "2026-06-15" AND ( (page_name='checkout_reservation' AND component_name='custom_option_add_button') OR (page_name='services_option' AND event_name='view') OR (page_name='services_option' AND component_name IN ('cta_button','product_detail_help')) ) GROUP BY step ORDER BY step -- 추가버튼 828, 옵션화면 785, 선택완료 578, 도움말 186
4. 일별 옵션 진입 이벤트 (앱 추가 버튼 / 옵션 화면 노출, 클릭)
일별 순 사용자 수
앱 추가 버튼은 checkout_reservation, 옵션 화면은 services_option
SELECT event_date_kst AS dt, page_name, event_name, component_name, COUNT(DISTINCT member_id) AS uu FROM `socar-data.app_web_log.socar_app_web_log` WHERE event_date_kst >= "2026-06-05" AND ( (page_name='checkout_reservation' AND component_name='custom_option_add_button') OR page_name='services_option' ) GROUP BY dt, page_name, event_name, component_name ORDER BY dt
5. 카시트 예약별 순결제액 분포 (단가 확인용)
예약 단위 순결제액 분포. 1만원 단위로 묶임(3만원 최다)
SELECT net_amount AS reservation_net_amount, COUNT(*) AS rsv_cnt FROM ( SELECT reservation_id, SUM(amount) AS net_amount FROM `socar-data.tianjin_replica.charged_info` WHERE DATE(created_at,"Asia/Seoul") >= "2026-06-05" AND charge_type = "addon_carseat" GROUP BY reservation_id HAVING net_amount > 0 ) GROUP BY reservation_net_amount ORDER BY reservation_net_amount
6. 마이그레이션 분리 검증 (카시트 마이그 시트 교집합)
카시트 마이그레이션 시트 예약번호 272개(고유)와 인앱 결제(charged_info)를 교집합

8. 결제 전환율 / 도움말 영향 / 매출 기여도

WITH paid AS ( SELECT DISTINCT ri.member_id FROM `socar-data.tianjin_replica.charged_info` ci JOIN `socar-data.tianjin_replica.reservation_info` ri ON ri.id = ci.reservation_id WHERE ci.charge_type IN ('addon_carseat','addon_stroller') AND DATE(ci.created_at,"Asia/Seoul") >= "2026-06-05" GROUP BY ri.member_id HAVING SUM(ci.amount) > 0 ), ev AS ( SELECT member_id, MAX(IF(event_name='view' AND REGEXP_EXTRACT(form_data,r'ps[\\":]+([a-z])')='o',1,0)) AS entered, MAX(IF(event_name='click' AND component_name='cta_button',1,0)) AS submitted, MAX(IF(event_name='click' AND component_name='product_detail_help',1,0)) AS saw_help FROM `socar-data.app_web_log.socar_app_web_log` WHERE event_date_kst BETWEEN "2026-06-05" AND "2026-06-15" AND page_name='services_option' GROUP BY member_id ) SELECT COUNTIF(e.submitted=1) AS submit_m, -- 578 COUNTIF(e.submitted=1 AND p.member_id IS NOT NULL) AS submit_paid, -- 227, 결제전환율 39.3% COUNTIF(e.saw_help=1 AND e.entered=1) AS help_enter, -- 186 COUNTIF(e.saw_help=1 AND e.entered=1 AND e.submitted=1) AS help_submit, -- 161, 86.6% COUNTIF(e.saw_help=0 AND e.entered=1) AS nohelp_enter, -- 599 COUNTIF(e.saw_help=0 AND e.entered=1 AND e.submitted=1) AS nohelp_submit -- 417, 69.6% FROM ev e LEFT JOIN paid p ON p.member_id = e.member_id
